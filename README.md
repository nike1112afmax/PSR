# PSR Product Spec Sheet Generator

A web-based tool for generating and comparing product specification sheets 
for ViewSonic EDU product lines.

## 🔗 Tool Link
https://nike1112afmax.github.io/PSR/PSR.html

## Features
- Product Spec Sheet (Single model or Compare up to 6 models)
- Logistic Information (up to 6 models)
- Compliance Certification Matrix (up to 6 models)
- Export to Word (.docx) and Print/PDF

## How to Use
1. Open the tool link above in Chrome or Edge
2. Upload the latest PSR Excel file
3. Select a product line tab and model
4. Edit and export
